# Policy Insight - AI 기반 국민 입법 수요 분석 플랫폼

## 🎯 프로젝트 개요

Policy Insight는 실시간으로 생성되는 뉴스 기사를 바탕으로 현행법령 문서를 활용하여 법안 제정, 개정, 폐지 수요를 예측하고 이슈의 중요도를 평가하는 AI 기반 플랫폼입니다.

**주요 사용자**: 국회, 정부 부처, 지자체 등 공공기관 종사자 및 정책 입안자

## 🎨 디자인 컨셉

- **색상**: 흰색과 검은색 기반의 전문적이고 신뢰감 있는 디자인
- **디자인 시스템**: KRDS(KoRea Design System) 표준 준수
- **UI 라이브러리**: shadcn/ui

## 🛠 기술 스택

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **상태 관리**: Zustand
- **스타일링**: Tailwind CSS
- **UI 컴포넌트**: shadcn/ui (Radix UI 기반)

## 🎪 주요 페이지

### 1. 인증 관련
- **로그인**: 이메일/비밀번호 로그인, 소셜 로그인 (구글, 네이버, 카카오)
- **회원가입**: 3단계 (약관 동의 → 정보 입력 → 완료)
- **아이디/비밀번호 찾기**: 이메일/전화번호 인증 방식

### 2. 대시보드
- **일간/주간/월간/연간 탭**: 시간대별 이슈 분석
- **주요 이슈 Top 10**: 
  - 이슈 제목 및 요약
  - 키워드 태그
  - 감정 분석 (긍정/중립/부정)
  - 관련 뉴스
  - 관련 법안 (제정/개정/폐지)
- **1순위 법안 상세**: 가장 시급한 입법 수요 강조

### 3. 리포트
- **리포트 목록**: 주간/월간 리포트 통합 뷰
- **리포트 상세**: 
  - 기간별 종합 분석
  - 주요 이슈 상세
  - 결론 및 제언
  - PDF 다운로드 기능

## 🎨 KRDS 디자인 원칙

1. **전문성**: 정부 서비스에 적합한 신뢰감 있는 디자인
2. **명확성**: 정보 계층 구조가 명확한 레이아웃
3. **접근성**: 모든 사용자가 쉽게 이용할 수 있는 인터페이스
4. **일관성**: shadcn/ui 기반 통일된 컴포넌트 시스템

## 🔐 인증 흐름

### 회원가입
1. 약관 동의 (이용약관, 개인정보 처리방침, 제3자 제공)
2. 정보 입력 (이메일/전화번호 인증 포함)
3. 회원가입 완료

### 로그인
- 일반 로그인 (이메일 + 비밀번호)
- 소셜 로그인 (구글, 네이버, 카카오)

### 계정 찾기
- 아이디 찾기: 이메일/전화번호 인증
- 비밀번호 찾기: 아이디 입력 + 인증 → 비밀번호 재설정
